# Security Policy

## Supported versions

This project is under active development. Security fixes target the `main` branch.

| Version | Supported |
|---------|-----------|
| `main`  | ✅ |

## Reporting a vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, use one of:

1. **GitHub Private Vulnerability Reporting** — go to the repository's
   **Security → Report a vulnerability** tab (preferred).
2. **Email** — `thqueirozsilva@gmail.com` with the subject `[SECURITY] stockr-api`.

Please include:

- A description of the vulnerability and its impact.
- Steps to reproduce (proof of concept if possible).
- Affected version / commit.

You'll receive an acknowledgement as soon as possible. Please give a reasonable
window for a fix before any public disclosure.

## Scope

This is a portfolio / educational project. There is no production deployment or
bug-bounty program, but responsible disclosure is appreciated and credited.
